Selected Projects

1. Urban Fringe Recreational Forest Park Construction

A research project related to urban forest park development, focusing on literature review, policy collection, and future trends of recreational forest parks in super-large cities.

2. Public Toilet Resource Scheduling Simulation

A simulation project based on cellular automata and particle swarm optimization, exploring resource scheduling for public toilets in high-density urban scenarios such as shopping malls and tourist attractions.

3. Shanghai Postal Museum Immersive Renovation Proposal

An immersive spatial renovation proposal for Shanghai Postal Museum. The project integrates postal culture, urban memory, cultural IP transformation, visitor experience design, and public cultural space renewal.

4. Ear-EEG Physiological State Sensing

A sensor-based project involving BLE communication, ear-EEG signal acquisition, CSV data cleaning, and exploratory applications in human-computer interaction and assistive control.
